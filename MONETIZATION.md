## Modelo de Ingresos Propuesto

### Tier 1: Free (Attraction)
- Test de salud básico
- Plan de ejercicios genérico
- Tracking manual
- **Objetivo**: Capturar usuarios, demostrar valor

### Tier 2: Premium ($9.99-19.99/mo)
- Planes personalizados por IA
- Análisis de comidas por foto
- Integración con wearables
- Proyecciones avanzadas de longevidad
- Chat con coach de IA

### Tier 3: Enterprise ($99-299/mo)
- Para gimnasios y clínicas
- Dashboard de clientes
- API de integración
- Reportes para pacientes
